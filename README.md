   ## Instruções para iniciar o Frontend

1. Na pasta <i>frontend</i>, dê esse comando:
   ~~~
   npm install
   ~~~
2. Após ter terminado, para iniciar a aplicação:
   ~~~
   npm run dev
   ~~~

## Instruções para iniciar o BackEnd

1. Na pasta <i>backend</i>, dê esse comando:
   ~~~
   npm install
   ~~~
    ~~~
   npx prisma init
   ~~~
2. Após ter terminado, para iniciar a aplicação:
   ~~~
   npm run start
   ~~~
